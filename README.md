Descrição
Repositório inicial TypeScript da estrutura Nest .

Instalação
$ npm install
Executando o aplicativo
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
Teste
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
